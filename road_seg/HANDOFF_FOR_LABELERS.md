# 道路面ラベル作成 協力者向け手順

## 目的

航空写真を見て「実際にトラックが走れる道路面・駐車場・私道」を緑マスクで修正し、教師データとして保存します。

## 必要なもの

- Windows
- Python 3.11 以上
- インターネット接続

## 起動

配布フォルダを展開して、`道路幅AIラベル作成.bat` をダブルクリックします。

初回は `.venv` を作成し、必要な Python パッケージを入れます。少し時間がかかります。

メニューが出たら:

1. `1) 動作確認` を実行
2. `4) 道路ラベル作成` を実行
3. ブラウザが開いたら、対象範囲へ移動して `この範囲を取得`
4. 緑の道路面を修正
5. `保存して教師データ化`

## 修正ルール

- 塗る: 実際に車両が走れる舗装面、施設内通路、駐車場内の走行面、私道
- 消す: 建物、屋根、芝生、畑、植え込み、川、水面、車両だけが写っている場所
- 迷う場所は無理に塗らず、明らかな走行面を優先
- 公道だけでなく、施設の入口から荷下ろし場所までの走行面が重要

## 保存先

保存すると、以下に3点セットで追加されます。

- `road_seg/dataset/images/<id>.png`: 航空写真
- `road_seg/dataset/masks/<id>.png`: 修正済み道路面マスク
- `road_seg/dataset/meta/<id>.json`: 範囲や編集情報

## 作業後に返すもの

作業が終わったら、`road_seg/dataset` フォルダをZIPにして返してください。

返却対象:

- `road_seg/dataset/images`
- `road_seg/dataset/masks`
- `road_seg/dataset/meta`

`road_seg/dataset/.pending` は返却不要です。

## 困った時

- 起動しない: Python が入っているか確認してください。
- 航空写真が出ない: インターネット接続、または地図を少し移動して再取得してください。
- 範囲が広すぎると言われる: ズームインしてから `この範囲を取得` を押してください。
- 保存できない: 起動している黒い画面を閉じずに、ブラウザを開いたまま操作してください。
