搬入経路の安全性を自動判定するシステム
1必要なもの
ソフトウェア 用途 種別
Python 3.8以上
python.org/downloads		 サーバー起動・各種処理 必須
Pythonパッケージ
fastapi / uvicorn / ultralytics / Pillow / requests		 AI解析サーバー（障害物検出） 必須
Google Maps APIキー
取得方法 		Street View・地図スナップショット 任意
⚠ Pythonインストール時は 「Add Python to PATH」にチェックを入れてください。
2初回セットアップ（一度だけ）
1
Pythonをインストール

python.org/downloads からダウンロード。インストール時に Add Python to PATH にチェック。

2
「初回セットアップ.bat」をダブルクリック

必要なPythonパッケージを自動インストールします。インターネット接続が必要です（数分かかる場合があります）。

4
「user_config.js」を編集（任意）

Google Maps APIキーや使用するAIモデルを変更できます。メモ帳で開いて編集してください。

3起動方法
1
「起動.bat」をダブルクリック

AIサーバー（ポート8001）とWebサーバー（ポート8080）が自動起動し、ブラウザが開きます。

2
ブラウザが自動で開かない場合

手動でアクセス: http://localhost:8080/index8.0.html

3
終了するには

起動.batのウィンドウを閉じるか Ctrl+C を押します。

4基本操作
1
住所で地図を移動

上部の検索ボックスに住所・施設名を入力して検索ボタンを押します。

2
経路を設定

地図上をクリックして搬入開始点・終点を2点追加します。クリックするたびに端点が追加され、経路が自動計算されます。

3
「搬入判定を実行」ボタンをクリック

道路幅・衝突・適合率を自動チェックし、スコアと判定結果を表示します。

4
「📄 レポート出力」で結果を保存

判定結果をA4サイズのHTMLレポートとして出力し、印刷またはPDF保存できます。

6トラブルシューティング
地図が表示されない

起動.batのウィンドウが開いていることを確認してください。ウィンドウを閉じるとサーバーが停止します。

Street Viewが表示されない

Google Maps APIキーが必要です。user_config.js の googleMapsApiKey に有効なキーを設定してください。

障害物検出が動かない

起動.batでYOLOサーバー（ポート8001）が起動していることを確認してください。「初回セットアップ.bat」でパッケージがインストールされているか確認します。
