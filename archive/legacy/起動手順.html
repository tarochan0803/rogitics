起動.bat をダブルクリック
以下の3つのサーバーが自動で起動し、ブラウザにアプリが開きます
  • Ollama（AI 所見サーバー）— port 11434
  • YOLO サーバー（画像解析）— port 8001
  • Web サーバー（アプリ配信）— port 8080

操作手順（5ステップ）
① 道路データ取得
サイドバー「WORKFLOW」の「🔄 道路データを取得」を押す。地図上の道路が読み込まれます。
② 車両選択
上部バーの車両カード（2t / 3t / 4t / 10t / 自由）から搬入車両を選択。
③ 経路設定
「📌 端点追加モードへ」を押し、地図をクリックして出発点・目的地を2点以上追加 → 「✅ 確定」。
④ 判定実行
「✅ 判定を実行する」を押す。YOLO・Ollama が利用可能であれば道路幅推定と AI 所見も自動実行されます。
⑤ 結果確認
画面左下の結果カードで PASS / CONDITIONAL / NG と詳細を確認。

トラブルシューティング
⚡ アプリが開かない / 「接続できません」
起動.bat を再度ダブルクリックしてください。すでに起動済みのサーバーはスキップされます。
⚡ YOLO サーバーが起動しない
初回セットアップ.bat を実行して依存ライブラリをインストールしてください。
アプリ内のサイドバー「SERVERS」パネルの「起動」ボタンからも再起動できます。
⚡ Ollama 所見が出ない
Ollama がインストールされているか確認し、ollama pull llava でモデルをダウンロードしてください。
アプリ内「SERVERS」パネルの「起動」ボタンで Ollama を起動できます。
⚡ ポートが競合している（他のアプリが使用中）
コマンドプロンプトで以下を実行してプロセスを確認してください。
netstat -ano | findstr ":8080 " netstat -ano | findstr ":8001 " netstat -ano | findstr ":11434 "
